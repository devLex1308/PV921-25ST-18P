Необхідно всі свої домашні роботи завантажити в репозиторій для цього.

1) Бути зареєстованим на сайті https://github.com
2) В особистому кабінеті є вкладка репозиторії (Your repositories) ідем туди.
Там є зелена кнопка, натискаємо придумуємо назву латинецею ставимо налаштування Private та натискаємо create repository
3) Переходимо в репозиторій натискаємо зелену кнопку code і копіюємо посилання типу https://github.com/devLex1308/PV921-25ST-18P.git
4) На своєму компютері відкриважмо в папці де ви хочите щоб був репозиторій git bush (права кнопка мишки git bush here). Пишемо git clone та вставляємо те що ви скопіювали та натискаємо ентер
5) Заходимо в репозиторій в консолі набираємо cd назва репозиторію (має писат master)
6) Створюємо свою вітку наприклад home-work
git checkout -b <назва вітки>
7) Вставляємо в репозиторій всі домашні роботи
8) Комітим
git add .
git commit -m "Якийсь змістовний коментар"
9) Скоріше за все спочатку треба ввести ім'я та пошту
git config --global user.name "Your Name"
git config --global user.email "your_email@whatever.com"

10) Запушити на сервер 
git push origin <назва вітки>

11) Піти цей репозиторій на сайті. Там є селект в лівому кутку де написано master натискаємо і вибираємо вітку яку ми запушили. Має з'явитись зелена кнопка create pull request. При створенні у Reviewers додати devLex1308
